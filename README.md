🎬 YouTube Clone

A fully responsive YouTube Clone built using HTML, CSS And JavaScript. This project replicates core features of YouTube like video browsing, search, and playback with a clean and user-friendly interface.

🚀 Features
🔍 Search videos
🎥 Video player with controls
📂 Categories / Sidebar navigation
🧾 Video details (title, views, channel info)
👍 Like / Subscribe UI (static or dynamic)
📱 Fully Responsive (Mobile + Desktop)
⚡ Fast loading UI

🛠️ Tech Stack
Frontend: HTML5, CSS3, JavaScript
Styling: CSS / Tailwind CSS

⚙️ Installation & Setup
1. Clone the repository:
git clone https://github.com/your-username/youtube-clone.git
2. Navigate to the project:
cd youtube-clone
3. Run the project:
For HTML/CSS/JS version:
Open index.html in browser
OR
Use Live Server (VS Code)
For React version:

npm install
npm start

🔑 API Setup (Optional)
If using YouTube API:
Go to Google Cloud Console
Create a new project
Enable YouTube Data API v3
Generate API Key
Add it in your project:
const API_KEY = "YOUR_API_KEY";
